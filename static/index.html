Welcome to Talkilla!

To find out more about Talkilla, please read the wiki page. Talkilla is under active development, please be aware that some features are currently incomplete.

Currently we are looking for feedback about specific issues:

  • Login / Browser set up issues
  • Issues starting conversations
  • Call Stability issues

We are happy to receive feedback via irc, email, bugzilla, direct conversation. You may also want to look at our Trello boards for the current and planned work.

To get started on Talkilla, click the button to add it to Firefox:

If you've already clicked it, you don't need to click it again, just look for the chat icon on your toolbar if the sidebar isn't displayed.

We're sorry, but your browser does not have the necessary support for Talkilla. Talkilla currently only supports Firefox 25 or higher - we plan to support all WebRTC browsers in the future.
